# Yashpal Academic Website

A responsive multi-page academic website designed for GitHub Pages.

## Website address

Repository name: `yashpal1506.github.io`

Public URL: `https://yashpal1506.github.io/`

## Files

- `index.html` — homepage
- `publications.html` — publications and manuscripts
- `achievements.html` — achievements, awards, and certifications
- `conferences.html` — conference and workshop activities
- `css/style.css` — design and responsive layout
- `js/main.js` — mobile navigation and subtle reveal effects
- `assets/profile.webp` — profile photograph
- `assets/Yashpal_Resume.pdf` — downloadable resume

## Publish using GitHub website

1. Open `https://github.com/yashpal1506/yashpal1506.github.io`.
2. Back up any existing files.
3. Upload all files and folders from this package to the repository root.
4. Commit the changes to the `main` branch.
5. Open **Settings → Pages**.
6. Under **Build and deployment**, choose **Deploy from a branch**.
7. Select `main` and `/ (root)`, then save.
8. After a few minutes, open `https://yashpal1506.github.io/`.

## Publish using Git

```bash
git clone https://github.com/yashpal1506/yashpal1506.github.io.git
cd yashpal1506.github.io
# Copy the package contents into this folder.
git add .
git commit -m "Create professional academic website"
git push origin main
```
